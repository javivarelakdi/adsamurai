# Box

<Description of={Box} />

Columns can figure out the layout on their own if set to `fill`:

<Canvas>
    <Story name="fill">
        {(args) => (
            <Box flex="row">
                <Box xsCol="fill">
                    <div style={{ background: "#ccc", height: "30px" }} />
                </Box>
                <Box xsCol="fill">
                    <div style={{ background: "#777", height: "30px" }} />
                </Box>
                <Box xsCol="fill">
                    <div style={{ background: "#ccc", height: "30px" }} />
                </Box>
            </Box>
        )}
    </Story>
</Canvas>

Also, if `fill` is set, they will adjust to the rest of the content:

<Canvas>
    <Story name="fill with other elements">
        {(args) => (
            <Box flex="row">
                <Box xsCol="fill">
                    <div style={{ background: "#ccc", height: "30px" }} />
                </Box>
                <div style={{ padding: "10px" }}>Content</div>
                <Box xsCol="fill">
                    <div style={{ background: "#ccc", height: "30px" }} />
                </Box>
            </Box>
        )}
    </Story>
</Canvas>
